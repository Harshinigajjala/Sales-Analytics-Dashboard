# 📊 Sales & Inventory Dashboard (Excel)

## 📖 Overview
This project contains an **Excel-based dashboard** that provides insights into:
- Customer segmentation by region and type
- Product category performance (Electronics, Furniture, Appliances, Office)
- Monthly sales vs. targets
- Profit margins and customer retention
- Inventory stock levels, reorder status, and dead stock analysis

The dashboard is designed to help businesses track **sales performance, profitability, and inventory health** in a structured and visual format.

---

## 📂 Project Structure
- **harshi_Dashboard.xlsx** → Main Excel file containing:
  - Customer data
  - Product catalog
  - Orders and sales transactions
  - Inventory details
  - Dashboard sheets 


## ✨ Features
- 📈 **Sales Analysis**: Monthly sales vs. targets, profit margins, and average order values  
- 👥 **Customer Insights**: Segmentation by region (North, South, East, West, Central),Repeat Customer,Top 10 Customers 
- 🛒 **Product Performance**: Category-wise contribution to sales and profitability  
- 📦 **Inventory Management**: Current stock, reorder levels, and dead stock tracking
